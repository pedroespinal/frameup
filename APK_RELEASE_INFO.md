## 📱 **INSTALLATION GUIDE**

### For Android Device (7.0+)
1. **Download** `frameup-v1.0.0.apk`
2. **Transfer** to Android device (USB or cloud)
3. **Settings** → Enable "Unknown Sources" (or allow from unknown apps)
4. **Open** APK file → Tap "Install"
5. **Grant Permission** → Location (optional, for Golden Hour)
6. **Launch** FrameUp and enjoy!

### System Requirements
- **Minimum:** Android 7.0 (API 24)
- **Recommended:** Android 10+ (API 29+)
- **Storage:** 100+ MB free space
- **Processor:** Modern ARM (ARMv7+, ARM64)
- **RAM:** 2GB+ recommended

### Permissions
```
✅ Location     - Optional (for Golden Hour GPS)
✅ Camera       - Optional (future photo features)
```
