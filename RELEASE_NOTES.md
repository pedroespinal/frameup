#### 🔧 Technical Tools (7)

1. **Exposure Triangle Calculator**
   - Aperture (f-stop) calculator
   - Shutter speed recommendations
   - ISO guidance

2. **Depth of Field (DOF) Calculator**
   - Hyperfocal distance
   - Near/far focus points
   - Sensor-specific calculations

3. **ND Filter Calculator**
   - Exposure time calculations
   - Filter combinations
   - Light stop reductions

4. **Lens Advisor**
   - Field of view calculator
   - Crop factor adjustments
   - Equivalent focal length

5. **EXIF Reader**
   - Metadata extraction
   - Shot settings review
   - Camera info parsing

6. **My Gear Manager**
   - Camera inventory
   - Lens library
   - Quick settings access

7. **ND Timer**
   - Long exposure countdown
   - Exposure tracking
   - Filter timing calculations

### System Requirements

**Minimum:**
- Android 7.0 (API 24) or iOS 13+
- 100 MB free storage
- Internet connection (for Golden Hour GPS)

**Recommended:**
- Android 10+ (API 29+) or iOS 15+
- GPS-capable device
- Camera app for reference

### Camera Permissions Required
- 📍 **Location** - Golden Hour calculator (requires "Allow while using")
- 📷 **Camera** - For future photo capture features
- 💾 **Storage** - For saving configurations and photos

### Known Limitations (v1.0.0)
- Firebase integration requires separate configuration
- Community feature requires backend setup
- Advanced EXIF editing not available in initial release
